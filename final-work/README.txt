Проект для самостоятельного проектирования.
База была задумана как доска задач (canban) в отдел разработки ПО.
База состит из таблицы пользователй с данными для авторизации и частью контактных данных.
Предусмотрено расширение данных пользователя за счет кастомных полей. Т.е. админ/пользователь сампридумывает поле данных. 
Пользователь или админ заполняет это поле соотвествующим значением. 

Есть команды разработчиков. Для команд предусмотрены роли, которые так же настраиваються (собственные для каждой команды).
Команды работают с проектами. Проекты состоят из задач.
К проектам и задачам можно прикрепить хэштэг.

Статусом задачи можно считать "дорожку" projectrow. У задачи и проекта есть общий аттрибут iteration, 
чтобы задачи исчезали с глаз, по кончанию итерации.
